Technical Summary

Overview of the monorepo boilerplate structure

1

Shared Types Package

Located in shared/. Export data models directly. Edits compile instantly across both workspaces.

2

Express API Server

Located in backend/. Built on Express and Node. Serves API endpoints and hosts the built SPA in production.

3

Angular SPA Client

Located in frontend/. A standalone client using modern features, HMR, and environment-synced dev server proxying.

Standardized Components

Interactive examples of the boilerplate's standard UI components

AG Grid Implementation

We use AG Grid for all complex data tables. This example demonstrates custom cell renderers (like the status badges), value formatters (like the % sign for CPU Load), built-in column filtering, and responsive sizing.

Apache ECharts (ngx-echarts) Implementation

We use Apache ECharts for all robust, interactive data visualizations. This example demonstrates a smooth area line chart custom-styled to integrate seamlessly with the project's glassmorphism and dark mode theme.
